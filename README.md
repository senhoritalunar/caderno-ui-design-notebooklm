# Caderno Temático - Fundamentos de UI Design

## Contexto

Este projeto foi desenvolvido como parte do desafio da DIO, com o objetivo de explorar o NotebookLM como ferramenta de aprendizagem ativa.

O tema escolhido foi **UI Design (User Interface Design)**, área responsável pela criação de interfaces digitais intuitivas, funcionais e visualmente organizadas. Durante o projeto, foram utilizadas fontes confiáveis para aprofundar o conhecimento sobre os princípios do UI Design e documentar todo o processo de estudo.

## Objetivos

- Compreender os conceitos fundamentais de UI Design.
- Identificar os princípios e boas práticas para criação de interfaces digitais.
- Diferenciar UI Design de UX Design.
- Aprender a utilizar o NotebookLM como ferramenta de estudo e organização do conhecimento.
- Consolidar os aprendizados em um miniguia de estudos.
